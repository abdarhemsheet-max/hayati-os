التشغيل محلياً

1
حمّل الكود

git clone https://github.com/abdarhemsheet-max/hayati-os.git

2
ثبّت الحزم وشغّل

npm install && npx prisma db push && npm run build && npm start

3
افتح المتصفح

على http://localhost:4400

النشر أونلاين (Vercel)

1
اربط المستودع مع Vercel

من dashboard.vercel.com → New Project → استيراد hayati-os

2
ضف المتغيرات البيئية

DATABASE_URL (PostgreSQL من Supabase)، SUPABASE_URL، SUPABASE_SERVICE_KEY، B2_KEY_ID، B2_APPLICATION_KEY، B2_BUCKET_NAME، B2_ENDPOINT

3
انشر!

Vercel سيبني وينشر تلقائياً — اضغط Deploy

مستودع GitHub التعليمات الكاملة انشر على Vercel
